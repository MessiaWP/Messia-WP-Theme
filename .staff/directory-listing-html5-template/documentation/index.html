How To Change Fonts

  1. If you are not satisfied with the default font, you can choose from hundreds of ones from the Google Fonts. After you are happy with the selected font, click SEE SPECIMEN. You'll be redirected to font detail, Then click SELECT THIS FONT. Popup window will show with text "1 Family Selected" - click it. Then you can customize your selection by clicking Customize. All necessary things are included at the popup. Or you can download the fonts and add font .ttf and .eot file at font folder then go on style.css file. At here @font-face code, you can include your fonts by adding it here. See More - https://css-tricks.com/dont-just-copy-the-font-face-out-of-google-fonts-urls/
  2. The final step is to change the font family for the <body>. Go back to Google Fonts page where you have chosen your font and find a Specify in CSS section. Then copy the highlighted text. In this example font-family: 'OpenSans';

 

How To Change The Colors

There is two method of how to change the main color. 1st Just by replacing the HEX color value in the style.css (simple) and 2nd by installing node.sj and PostCSS (advanced). Here is a simple way.

Let say, that you want to change the default red color to blue one:

  1. Open the style.css in css/ folder
  2. Find the HEX value of the Default Color. If the default color is red, then HEX value is #ff0000
  3. In your text editor find Replace or Find and Replace function
  4. As original text enter the red color #ff0000 (template default color)
  5. As Replace with entering HEX value of blue color #0000ff

Owl Carousel

This plugin is used for creating slider/carousels from list of images or even HTML elements. Carousel must be wrapped in owl-carousel class. 

Example: Carousel with 3 visible slides, 30px margin, auto play, infinite loop and 5s pause.

<div class="owl-carousel" data-owl-items="3" data-owl-margin="30" data-owl-autoplay="1" data-owl-loop="1" data-owl-autoplay-timeout="2000">
<img src="assets/img/1.jpg">
<img src="assets/img/2.jpg">
<img src="assets/img/3.jpg">
</div>

 

All data-* attributes you can use with owl-carousel
  1. data-owl-items number of items in the carousel. 1 item for fullscreen carousel
  2. data-owl-nav show or hide navigation arrows. Default: 0
  3. data-owl-dots show or hide navigation dots. Default: 0
  4. data-owl-center center carousel. Default: 0
  5. data-owl-loop infinite loop. Default: 0
  6. data-owl-auto-width carousel will move according to the width of the slides. Default: 0
  7. data-owl-nav-container container for external navigation. Default: 0
  8. data-owl-autoplay carousel will play automatically. Default: 0
  9. data-owl-autoplay-timeout pause between slides in milliseconds. Default: 5000
  10. data-owl-auto-height height of the carousel will automatically change according to the slide height. Default: 5000
  11. data-owl-fadeout fade effect between transitions. Default: 0
  12. data-owl-extranav  if you want extra navigation control then you need to set it 1, Default: 0
  13. data-owl-indicator it's for extra indicator control, Default: 0

 

Google Maps

You have to obtain your unique API key for the Google Map, otherwise, you can see this error: 

  1. Go to https://developers.google.com/maps/documentation/javascript/get-api-key#get-an-api-key
  2. Click Get Key
  3. After you have successfully obtained a key, open an HTML page with your map
  4. Scroll to the very bottom where the <script> tags are
  5. Find <script src="http://maps.google.com/maps/api/js?key=YOUR_API_KEY&amp;Fallback"></script>
  6. Replace YOUR_API_KEY with your new API key

 

We include some fallback script at function JS. So after API key, you can put your callback style of google maps
